# Contributing

See [CLAUDE.md](./CLAUDE.md). It's the contributing guide for everyone (humans and AI agents) and covers the issue-first policy, writing rules, repo structure, PR checklist, and license terms.

The contribution we'd most love right now: **feature-parity plugins for other coding agents** (Cursor, OpenCode, etc.). The skills themselves are just markdown, so the work is wrapping them so they activate in those harnesses the way they do in Claude Code.
